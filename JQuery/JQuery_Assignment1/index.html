.HTML

The html method allows you to change the text content of an element with the addition of using html methods.

.text

.text method is essentially the same as the .html method. The difference being is that the .text method does not allow you to use any sort of .html methods.

.val

The .val method allows one to return value attributes from the first matched element. It also allows one to set the value attribute using a function. This particular function is very useful for the "input" method.

.hide .show .fadeOut .fadeIn

The .hide and .show methods hide and shows elements. It can be useful for thing such as online study tools. The fadeOut and fadeIn are very similar to the .hide .show method and dunction essentially the same.
